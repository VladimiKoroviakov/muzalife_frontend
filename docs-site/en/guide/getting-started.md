# Getting Started

## Requirements

| Tool    | Version |
|---------|---------|
| Node.js | 20.x    |
| npm     | 10+     |

## Installation

```bash
git clone https://github.com/your-org/muzalife-frontend.git
cd muzalife-frontend
npm install --legacy-peer-deps
```

## Running

```bash
npm run dev        # dev server at https://localhost:5173
npm run build      # production build
npm run type-check # TypeScript check
```

## Documentation

```bash
npm run docs          # TypeDoc → docs/typedoc/
npm run test:docs     # living documentation tests
npm run docs:archive  # create docs/typedoc.zip
```
